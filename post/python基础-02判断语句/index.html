Python基础-02判断语句

1. 判断语句

1.1 if/else

基本语法

if 要判断的条件：
    条件成立时，要做的事情
else:
    条件不成立时，要做的事情
  • 代码缩进为4个空格
  • 将if和else共同缩进的代码看成一个代码块
  • 根据左侧箭头可以看到代码是否在代码块内部，根据光标也可以显示

1.2 if/elif

  • 同时判断多个条件，所有条件之间是平级的
  • 基本语法
if 条件1：
    条件1满足时，执行的代码
   ……
elif 条件2：
    条件2满足时，执行的代码
elif 条件3：
    条件3满足时，执行的代码
……
else：
    条件都不满足时，执行的代码

1.3 if的嵌套

  • 判断的条件之间有前后关系
  • 基本语法
if 条件1：  
    条件1满足时，执行的代码  
    ……  
    if 条件1基础上的条件2：  
        条件满足时，执行的代码  
        ……
    else：  
        条件不满足时，执行的代码  
        
else： 
    条件1不满足时，执行的代码  
    ……

# 导入随机工具包
import random

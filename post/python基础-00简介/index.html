一、Python简介

Life is short,you need python!

创始人：Guido Van Rossum(吉多 范罗苏姆)
时间：1989
1991:python解释器诞生，c语言，开源
1999，基金申请，computer programming for everybody

  • 目标：简单直观、开源、纯英语一样理解、短期开发
  • 设计哲学：优雅、明确、简单 用一种方法做一件事情!
  • 特点：完全面向对象、强大的标准库、大量的第三方模块（使用方式与标准库类似，覆盖科学计算、人工智能、机器学习、web开发、数据库接口、图形系统）
  • 面向对象的思维方式：找一个有能力的对象帮我们完成工作！
  • 可扩展性：用C，C++编写核心程序，在python中使用它们。
  • 缺点：运行速度不够快；国内市场小；中文资料匮乏
编译型语言 解释性语言
c,c++
编译器 解释器
速度快 翻译一行，执行一行；跨平台能力强

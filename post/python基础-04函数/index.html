1. 简介

    • 调用函数–享受封装的成果

  • F7 step into 单步执行，如果是函数，会进入函数内部
def multiple_table():
    # 打印九九乘法表
    """
    1*1=1
    1*2=2   2*2=4
    1*3=3   2*3=6   3*3=9
    """
    row = 1
    
    while row <= 9:
    
        col = 1
        while col <= row:
            print("%d * %d = %d" % (col, row, row * col), end="\t")
            col += 1
    
        print("")
        row += 1

# 函数实例
name = "小明"

def say_hello():
    print("Hello 1")
    print("Hello 2")
    print("Hello 3")

2.2 函数的文档注释

  • 如果希望给函数添加注释，应该注释在 定义函数的下方， 使用连续的三对引号
  • 在连续的三对引号之间编写对函数的说明文字
  • 在函数调用位置，使用快捷键crtl+Q可以查看函数的说明信息

注意：函数体相对比较独立，函数定义的上方，应该和其他代码（包括注释）保留2个空行

2.3 函数的参数

  • 在函数名的后面的小括号内部填写参数
  • 多个参数之间使用，分隔
def sum_2_num(num1, num2):
    """
    形参
    对两个数字求和
   
    """
    result = num1 + num2

    print("%d + %d = %d" % (num1, num2, result))


sum_2_num(50, 20) #实参

50 + 20 = 70

形参和实参

  • 形参：定义函数时，小括号中的参数，是用来接收参数的，在函数内部作为变量使用
  • 实参：调用函数时，小括号中的参数，是用来把数据传递到函数内部的

2.4 函数的返回值

  • 在函数中使用return关键字可以返回结果
  • 调用函数一方，可以使用变量来接受函数的返回结果

注意：Return表示返回，后续的代码都不会被执行

2.5 函数的嵌套调用

  • 一个函数里面又调用了另外一个函数，这就是函数的嵌套调用
# 函数的嵌套调用
def test1():
    print("*" * 50)


def test2():
    """函数的嵌套调用"""

    print("-" * 50)
    test1()
    print("+" * 50)


test2()

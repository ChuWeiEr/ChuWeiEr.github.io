1、axes画子图–以下两例都是对象操作

  • figure\axes\subplot\axis 关系和区别

  • axes的用法和subplot是差不多的，四个参数

  • 前两个指的是相对于坐标原点的位置，后两个指的是坐标轴的长/宽度，

import numpy as np
import matplotlib.pyplot as plt

# 1.创建一个画板
fig = plt.figure()
# 2.创建了一个轴
ax = fig.add_subplot(111)
# 3.所以就算只有一个子图，也可以生成一个subplot
# 然后对这个subplot对象进行各种轴、标注、刻度等的设定
ax.set(xlim=[0.5,4.5], ylim=[-2,8], title="An example of axes", xlabel="x-axis",
       ylabel="y-axis")

plt.show()

2. Axes方法与pyplot函数

第一种方法-函数

  • 调用了pyplot中的plot（）函数和xlim（）函数

第二种方法-对象

  • 使用了生成的subplot对象的两种方法：.plot和.set_xlim方法

实际上，这也是贯穿整个python编程的两种思路，函数式编程和对象式编程

  • 以plot()为代表的的函数式操作，表达简洁，但是没有体现出真正画图的实现过程，例如甚至当没有搞清楚Figure、Axes、Subplot等概念的时候，依然可以轻松地用pyplot函数画图，当子图较多的时候，对子图的操作容易混乱。
  • 以.plot为代表的的对象式操作，表达明确，分步写成Figure和Axes、subplot，操作过程可以看出是在哪张子图上操作，缺点是，需要写的代码较多，不够简洁。
# *-* coding:utf8 *-*
import numpy as np
import matplotlib.pyplot as plt

#

plt.xlim(0.5, 4.5)

fig2 = plt.figure(2)

ax.plot([1, 2, 3, 4], [10, 20, 30, 40], color="#00ffff", linewidth=3)
ax.set_xlim(0.5, 4.5)

plt.show()

  • contour
  • contourf():带填充效果的等值线!
# *-* coding:utf8 *-*
import numpy as np
import matplotlib.pyplot as plt


def f(x, y):
    return (1-x/2+x**5+y**3)*np.exp(-x**2-y**2)


n = 256
x = np.linspace(-3, 3, n)
y = np.linspace(-3, 3, n)
X, Y = np.meshgrid(x,y)
# 理解meshgrid的工作机制
# Y2, X2 = np.meshgrid(y,x)
# print("x = ", x)
# print("y = ", y)
# print("X = ", X)
# print("Y = ", Y)
# print("X2 = ", X2)
# print("Y2 = ", Y2)

# axes的用法和subplot是差不多的，四个参数
# 前两个指的是相对于坐标原点的位置，后两个指的是坐标轴的长/宽度，
# 可以看看这个：https://www.zhihu.com/question/51745620
plt.axes([0, 0, 1, 1])
plt.contourf(X,Y,f(X,Y),8,alpha=0.75,cmap=plt.cm.Greens)

C = plt.contour(X, Y, f(X,Y), 8, colors='black', linewidth=.5)
plt.clabel(C, inline=1, fontsize=10)

plt.show()

隐函数的等值线图

# *-* coding:utf8 *-*
# 使用等值线绘制隐函数曲线
# 可以使用等值线解决这个问题，
# 显然隐函数的曲线就是值等于0的那条等值线。
import numpy as np
import matplotlib.pyplot as plt

y, x = np.ogrid[-1.5:1.5:200j, -1.5:1.5:200j]
f = (x**2 + y**2)**4 - (x**2 - y**2)**2
plt.figure(figsize=(9, 4))
plt.subplot(121)
extent = [np.min(x), np.max(x), np.min(y), np.max(y)]
# 在调用contour()绘制等值线时，可
# 以通过levels参数指定所绘制等值线对应的函数值，
# 这里设置levels参数为[0，0.1],因此最终将绘制两条等值线。
cs = plt.contour(f, extent=extent, levels=[0, 0.1],
                 colors=["b", "r"], linestyles=["solid", "dashed"], linewidths=[2, 2])

plt.subplot(122)
# cs对象的collections属性是一个等值线列表，
# 每条等值线用一个LineCollection对象表示：
for c in cs.collections:
    data = c.get_paths()[0].vertices
    plt.plot(data[:, 0], data[:, 1],
             color=c.get_color()[0], linewidth=c.get_linewidth()[0])

plt.show()

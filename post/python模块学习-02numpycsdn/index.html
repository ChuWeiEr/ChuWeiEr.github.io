1.2 数组的切片

  1. numpy切片时产生新的视图，不是副本

1.4 数组的拼接/分裂

  • 二维数组的轴：竖轴为第一个轴，索引值为0；横轴为第二个轴，索引值为1

拼接

  • numpy.concatenate:可以指明拼接的轴；默认为0轴，是行数增加
  • numpy.hstack：横轴；等价于axis = 1
  • numpy.vstack：竖轴
  • numpy.dstack：第三个轴

分裂

  • np.split: 默认沿着axis=0（竖轴）分裂
  • np.hsplit:沿着横轴分裂
  • np.vsplit：沿着竖轴分裂
# 拼接 concatenate

print("a3 = ", a3)
# axis = 1
a4 = np.concatenate([a1,a2],axis=1)
print("a4 = ", a4)
print("a4.shape = ", a4.shape)

2.5 聚合

第一部分

  • np.sum()
  • np.prod()：积
  • np.mean()
  • np.std()：标准差
  • np.var()：方差
  • np.min()
  • np.max()
  • np.argmin()：最小值的索引
  • np.argmax()
  • np.median()：中位数
  • np.any()：是否至少存在一个为真的元素
  • np.all()
  • np.sort():排序，不修改数据源
  • np.argsort(): 返回排序后的索引
  • np.partition():中间值在中间，但是左右都不排序

第二部分

  • ufunc函数本身还有些方法，这些方法只对两个输入一个输出的ufunc函数有效
  • 其它的ufunc对象调用这些方法时会抛出ValueError异常。
  • reduce
  • reduce 方法和Python的reduce函数类似，它沿着axis轴对array进行操作.
  • accumulate
  • accumulate 方法和reduce方法类似，只是它返回的数组和输入的数组的shape相同，保存所有的中间计算结果：
# 聚合操作
# reduce
import numpy as np
a = np.arange(1,10)
print(" a= ", a)
# 把a内所有值相加
b = np.add.reduce(a)
print("b = ", b)
# 把a内所有值相加,并保存相加的中间结果
c = np.add.accumulate(a)
print("c = ", c)

# sum求和
d = np.sum(a)
print("d = ", d)

# 求最值
e = np.min(a)
f = np.max(a)

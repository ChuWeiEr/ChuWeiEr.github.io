3、变量

  • 在内存中分配的空间
  • 程序是用来处理数据的，变量是用来存储数据的
  1. python解释器根据赋值语句等号右侧的数据自动推导变量中保存数据的准确类型 str：字符串 int：整数 bool:布尔，True或者False float：小数，浮点数

3.1 数据类型分为 数字型和非数字型

  • 数字型

    • 整型（int）
    • 浮点型（float）
    • 布尔型（bool） 非零即真;可以参与计算，True = 1；False = 0.
    • 复数型（complex）

  • 非数字型

    • 字符串
    • 列表
    • 元组
    • 字典

  • 查看类型

    • type(name)

3.2 变量之间的计算

  • 数字型之间直接计算
  • 字符串之间可以使用 + 拼接生成新的字符串，使用*重复拼接相同的字符串
firstname = 'chu'
lastname = 'wei'
firstname + lastname

'chuwei'

(firstname + lastname) * 10

'chuweichuweichuweichuweichuweichuweichuweichuweichuweichuwei'

# 数字型变量和字符串之间不能进行其他计算
firstname + '10'

3.3 变量的输入

  1. 用户输入的任何内容python都认为是一个字符串
    字符串变量 = input(“提示信息”）
passwd = input("请输入银行密码")

type(passwd)

type(int('123'))

int
  1. 通过变量转换输入整数
price_str = input("苹果的单价： ")
weight_str = input("苹果的重量： ")
price = float(price_str)
weight = float(weight_str)
money = price*weight
print(money)

苹果的单价： 4.5
苹果的重量： 5.5
24.75
  1. 定义浮点变量接收用户输入时，直接用float转换 用户输入
price = float(input("苹果的单价： "))
weight = float(input("苹果的重量： "))
print(price*weight)

苹果的单价： 12
苹果的重量： 5
60.0

3.4 变量的格式化输出

  1. 语法格式
    print(“格式化字符串”%变量1)
    print(“格式化字符串”%（变量1，变量2……）)
格式化操作符 含义
%s 字符串
%d 有符号十进制整数，%06d表示输出的整数显示位数，不足的地方使用0补全，够了原样输出
%f 浮点数，%.2f表示小数点后只显示两位
%% 输出% 
name = "大明"
print("我的名字叫%s,请多多关照！"%name)
studentID = 11237896586592
print("我的学号是%06d"%studentID)

我的名字叫大明,请多多关照！
我的学号是11237896586592

price = float(input("苹果的单价： "))
weight = float(input("苹果的重量： "))
money = price*weight
print("苹果单价%.2f 元/斤, 购买了%.2f斤，需要支付%.2f元"%(price,weight,money))

苹果的单价： 12
苹果的重量： 45
苹果单价12.00 元/斤, 购买了45.00斤，需要支付540.00元

scale = 0.5
print("数据比例是 %.2f%%"%(scale*100))

数据比例是 50.00%

3.5 变量的命名

  1. 标识符可以由字母、下划线、数字组成
  • 不能以数字开头
  • 不能与关键字重名（Python内部已经使用的标识符）
  • python的标识符区分大小写
  • 多个单词组成，都用小写，单词之间使用下划线链接,eg: first_name

3、变量

  • 在内存中分配的空间
  • 程序是用来处理数据的，变量是用来存储数据的
  1. python解释器根据赋值语句等号右侧的数据自动推导变量中保存数据的准确类型 str：字符串 int：整数 bool:布尔，True或者False float：小数，浮点数

3.1 数据类型分为 数字型和非数字型

  • 数字型

    • 整型（int）
    • 浮点型（float）
    • 布尔型（bool） 非零即真;可以参与计算，True = 1；False = 0.
    • 复数型（complex）

  • 非数字型

    • 字符串
    • 列表
    • 元组
    • 字典

  • 查看类型

    • type(name)

3.2 变量之间的计算

  • 数字型之间直接计算
  • 字符串之间可以使用 + 拼接生成新的字符串，使用*重复拼接相同的字符串
firstname = 'chu'
lastname = 'wei'
firstname + lastname

'chuwei'

(firstname + lastname) * 10

'chuweichuweichuweichuweichuweichuweichuweichuweichuweichuwei'

# 数字型变量和字符串之间不能进行其他计算
firstname + '10'

3.4 变量的格式化输出

  1. 语法格式
    print(“格式化字符串”%变量1)
    print(“格式化字符串”%（变量1，变量2……）)
格式化操作符 含义
%s 字符串
%d 有符号十进制整数，%06d表示输出的整数显示位数，不足的地方使用0补全，够了原样输出
%f 浮点数，%.2f表示小数点后只显示两位
%% 输出% 
name = "大明"
print("我的名字叫%s,请多多关照！"%name)
studentID = 11237896586592
print("我的学号是%06d"%studentID)

我的名字叫大明,请多多关照！
我的学号是11237896586592

price = float(input("苹果的单价： "))
weight = float(input("苹果的重量： "))
money = price*weight
print("苹果单价%.2f 元/斤, 购买了%.2f斤，需要支付%.2f元"%(price,weight,money))

苹果的单价： 12
苹果的重量： 45
苹果单价12.00 元/斤, 购买了45.00斤，需要支付540.00元

scale = 0.5
print("数据比例是 %.2f%%"%(scale*100))

数据比例是 50.00%

第二部分、笔记

  1. plt的Rectangle参数：
    第一个参数是坐标（x，y），即矩形的画图的起点坐标，这个起点坐标不是一味地从左下角开始画，而是对应整个图中坐标原点，即（0，0）
    第二个参数是矩形宽度
    第三个坐标是矩形高度
    当前的图表和子图可以使用plt.gcf()和plt.gca()获得，分别表示Get Current Figure和Get Current Axes。
    在pyplot模块中，许多函数都是对当前的Figure或Axes对象进行处理，比如说：plt.plot()实际上会通过plt.gca()获得当前的Axes对象ax，然后再调用ax.plot()方法实现真正的绘图。
  2. mgrid
    这里的三个变量分别是k（x轴）、b（y轴）以及ErrorArray（z轴）
    mgrid中第三个参数越大，说明某一区间被分割得越细，相应的曲面越精准。
  3. alpha表示透明度；facecolor=‘none’表示不填充
  4. sys模块的常见函数列表
    sys.argv: 实现从程序外部向程序传递参数。
    sys.exit([arg]): 程序中间的退出，arg=0为正常退出。
    sys.getdefaultencoding(): 获取系统当前编码，一般默认为ascii。
    sys.setdefaultencoding(): 设置系统默认编码，执行dir（sys）时不会看到这个方法，在解释器中执行不通过，可以先执行reload(sys)，在执行 setdefaultencoding(‘utf8’)，此时将系统默认编码设置为utf8。（见设置系统默认编码 ）
    sys.getfilesystemencoding(): 获取文件系统使用编码方式，Windows下返回’mbcs’，mac下返回’utf-8’.
    sys.path: 获取指定模块搜索路径的字符串集合，可以将写好的模块放在得到的某个路径下，就可以在程序中import时正确找到。
    sys.platform: 获取当前系统平台。
    sys.stdin,sys.stdout,sys.stderr: stdin , stdout , 以及stderr 变量包含与标准I/O 流对应的流对象. 如果需要更好地控制输出,而print 不能满足你的要求, 它们就是你所需要的. 你也可以替换它们, 这时候你就可以重定向输出和输入到其它设备( device ), 或者以非标准的方式处理它们

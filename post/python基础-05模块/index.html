Python基础-05模块

1. 模块

  • 模块好比是一个工具包，需要导入import这个模块
  • 每一个以扩展名py结尾的python源代码文件都是一个模块
  • 在模块中定义的全局变量、函数都是模块能够提供给外界使用的工具

模块是python程序架构的一个核心概念

# 模块  cw_10_module
# 注意：模块名也是一个标识符


def print_line(char,times):
    print(char * times)


def print_lines(char,num):
     """
     打印多行分割线
     :param char:字符
     :param num:次数
     :return:
     """
     row = 0

     while row < 5:

         print_line(char, num)
         row += 1

name = "楚天阔"

# 导入模块
import cw_10_module

cw_10_module.print_line("-", 50)
print(cw_10_module.name)

.pyc文件

  • py是python的简写
  • c 是compile编译过的意思
    操作步骤
  • 浏览程序目录会发现一个_pycache_的目录
  • 目录下的cw_10_module.cpython-36.pyc文件，cpython-36表示python解释器的版本
  • 这个pyc文件是由python解释器将 模块的源码 转换成 字节码
  • python这样保存字节码是作为一种启动速度的优化

字节码

  • python解释源程序分为两个步骤
    1. 首先处理源代码，编译生成一个二进制的 字节码
    • 对字节码进行处理，才会生成cpu能够识别的机器码
